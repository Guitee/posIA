# Projeto de aula - TensorFlow.js

Este projeto é um exercício do modulo 2 da pos de ia de uso do TensorFlow.js em Node.js para treinar e testar um modelo de classificação.

## Versão do Node
- Node.js 20.14.0

## Como instalar
Na pasta do projeto, execute:

```bash
npm install
```

## Comandos usados para instalar o TensorFlow

```bash
npm install @tensorflow/tfjs
```

## Ajuste no Windows
Em alguns ambientes do Windows, o TensorFlow.js pode dar problema por causa da configuração do ambiente ou de dependências do sistema.
:

```bash
npm start
```

## Comando para iniciar o projeto

```bash
npm start
```

## Observações
- O arquivo principal é index.js, dentro da pasta do exemplo.
